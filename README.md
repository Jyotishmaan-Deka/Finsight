# Finsight

A personal finance app for Android.

## Features

- See your balance and monthly spending on the home screen
- Add, edit, and delete transactions
- Set savings goals, no-spend challenges, or budget limits
- View spending charts and trends
- Dark mode support

## Tech

Kotlin, Jetpack Compose, Room, Hilt, Material 3
